# Internship Task Plan

## Internship Domain

Full-Stack Development

## Selected Project

Task Management System

## Main Goal

Build a complete full-stack application that demonstrates REST API development, frontend integration, database usage, authentication, authorization, and deployment.

## Planned Work

### Level 1 - Basic

* Set up the development environment
* Create a simple REST API using Node.js and Express.js
* Build a basic frontend that interacts with the REST API

### Level 2 - Intermediate

* Build the frontend using React
* Add user authentication and authorization
* Integrate MongoDB database with the backend

### Level 3 - Advanced

* Build a complete full-stack application
* Connect frontend, backend, database, and authentication
* Prepare deployment
* Record a final project demo video
* Submit GitHub repository and project proof

## Project Features

* User registration and login
* Create tasks
* View tasks
* Update tasks
* Delete tasks
* Protect user-specific task data
* Role-based access preparation

## Development Approach

The project will be built step by step. Each major feature will be tested, committed to Git, and documented before moving to the next feature.
