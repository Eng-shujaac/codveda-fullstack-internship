# Codveda Full-Stack Development Internship

This repository contains my Full-Stack Development Internship tasks for Codveda Technologies.

## Internship Domain

Full-Stack Development

## Project Title

Task Management System

## Project Description

The Task Management System is a full-stack web application designed to help users create, manage, update, and track tasks.

This project is being developed as part of my Codveda Full-Stack Development Internship to practice and demonstrate real-world skills in frontend development, backend development, REST API design, database integration, authentication, authorization, and deployment.
